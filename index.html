GitLab - Ubuntu

GitLab-Ubuntu provides installation and packages for GitLab on Ubuntu. Ubuntu server is the intended target, as all instructions are given using the command line, but the scripts should function just the same on a desktop install of Ubuntu.

Install Scripts

Until a deb package can be created, the install script under source/install-stable.sh must be used.

Supported Versions

GitLab

Tested with version 3.03 www.gitlabhq.com

Ubuntu

Tested with Ubuntu Server 12.04.1 LTS x86_64
